# Research 模块记忆

## 当前定位

- `research` 已升级为 `project` 中的第三个正式模块
- 它负责把事实搜集、来源区分和证据交接标准化

## 长期约束

- research 结果必须区分来源和推断
- research 输出不等于成稿
- research 与 review、verification 协作，但职责不同

## 维护规则

- shared 侧维护 workflow / skill / playbook / memory
- Claude / Codex 侧只做适配层
- 研究结果应尽量能被 writing、review、verification 复用
