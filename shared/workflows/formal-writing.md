# Formal Writing Workflow

## Purpose

Provide a reusable workflow for formal drafting, rewrite, polish, and formatting handoff.

## Typical Inputs

- document type
- audience
- source facts
- required unchanged wording
- deadlines and actors
- preferred output format

## Typical Outputs

- structured draft
- internal briefing
- polish pass
- formatting handoff package

## Capability Lanes

- writing
- review
- research when policy or reference material matters

## Suggested Shared Skill Route

- `internal-comms`
- `official-doc-format`
- `humanizer-zh`

Use the example in `examples/formal-writing/README.md` to see how Claude and Codex adapt this workflow differently.
