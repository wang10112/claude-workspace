# Internal Communications

## When To Use

Use for internal reports, briefings, status notes, and explanation memos.

## Core Responsibility

Turn rough facts into a restrained, decision-friendly internal document.

## Input Expectations

- goal or request
- facts and timeline
- actors, actions, and deadlines

## Output Expectations

- key point first
- high fact density
- non-promotional tone
- use MECE structure where helpful so sections are mutually exclusive and collectively exhaustive

## Boundary Rules

- do not invent facts
- do not use it as a formatting-only step
- avoid repeating the same point across adjacent paragraphs or lists
