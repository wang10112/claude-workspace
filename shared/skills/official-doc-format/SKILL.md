# Official Document Formatting

## When To Use

Use when wording is already approved and layout is the main task.

## Core Responsibility

Normalize text or document structure into an official-style layout without rewriting substance.

## Input Expectations

- approved body text
- target document type
- date, title, and addressee data if required

## Output Expectations

- disciplined structure
- title and closing placement
- no content invention

## Boundary Rules

- if the source draft is weak, route back to writing before formatting
