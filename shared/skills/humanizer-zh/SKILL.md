# Humanizer ZH

## When To Use

Use after structure and facts are already correct.

## Core Responsibility

Make Chinese prose sound more natural and experienced without changing the underlying substance.

## Input Expectations

- a structurally correct draft
- clarity on which wording must remain unchanged

## Output Expectations

- reduced repetition
- less templated transitions
- preserved meaning

## Boundary Rules

- do not use as a substitute for fact checking
- do not use when exact wording must be preserved everywhere
