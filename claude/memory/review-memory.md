# Review 模块记忆

## 当前定位

- `review` 已升级为 `project` 中的第二个正式模块
- 它不只是一个通用动作词，而是共享层中的正式能力面

## 长期约束

- review 输出始终以 findings-first 为默认结构
- review 与 verification 相关但不等同
- review 更偏分析判断，verification 更偏 readiness 与已执行检查

## 维护规则

- shared review 定义优先放在 `claude/workflows/`、`claude/skills/`、`claude/playbooks/`
- Claude / Codex 侧只做适配，不复制第二份正文
