# Claude Hook: pre-write-check

## Purpose

Make sure drafting starts with the correct skill route instead of jumping directly into uncontrolled output.

## Trigger

Before drafting or rewriting substantial content.

## Suggested Action

- identify whether the task is drafting, formatting, polish, review, or research
- route to the right Claude skill
- confirm whether exact wording must be preserved

## Runtime Role

This is primarily a learning sample, but it can map to a real Claude write-guardrail flow.
