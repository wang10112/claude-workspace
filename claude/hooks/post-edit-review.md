# Claude Hook: post-edit-review

## Purpose

Encourage a review pass after meaningful content or structure edits.

## Trigger

After substantial edits to shared assets, Claude adapters, or module docs.

## Suggested Action

- call the review lane
- check for drift against shared sources
- verify whether the edit changed module routing expectations

## Runtime Role

This is both:

- a learning sample
- a blueprint for a future Claude review-oriented hook
