# Document Polish Workflow

## Purpose

Refine an already correct draft without changing its factual substance.

## Focus

- remove stiffness
- improve readability
- preserve structure and facts
- prepare for formatting if needed

## Typical Shared Skill Route

- `formal-writing` - 用于正式文档的迭代优化
- `writing` - 用于简单文档的快速润色
- optional `humanizer-zh` - 仅在需要去除 AI 味道时使用（会降低可读性）
