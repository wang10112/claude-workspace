# Claude Skill: writing

## Purpose

This is the Claude-facing writing skill for structured drafting and rewriting tasks across modules.

## When To Use

- draft a new artifact
- rewrite an existing artifact
- turn source notes into a readable output

## Input Expectations

- audience
- goal
- source material
- tone or style constraints

## Output Expectations

- clear structure
- audience-aware language
- no invented facts

## Shared Relationship

This skill bridges general writing work to shared workflows and module-specific skills.

## Collaboration Pattern

- pair with `agents/writer.md` for drafting
- pair with `skills/formal-writing/` for the built-in formal-writing module
- pair with `skills/review/` after a draft exists
