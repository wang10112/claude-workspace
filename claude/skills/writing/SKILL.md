# Writing

## When To Use

Use for new drafts, rewrites, and note-to-document transformation.

## Core Responsibility

Produce clear, audience-aware writing from source material without inventing facts.

## Input Expectations

- audience
- goal
- source material
- tone or style constraints

## Output Expectations

- clear structure
- readable wording
- audience-fit framing
- preserved factual meaning
- use MECE structure where it improves clarity and reduces overlap

## Boundary Rules

- do not invent missing facts
- if the task is formatting-only, route to formatting instead of rewriting
- avoid stacking partially overlapping categories, paragraphs, or request items
