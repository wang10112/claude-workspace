# Claude Skill: research

## Purpose

This is the Claude-facing research skill for source collection, fact separation, and synthesis before writing or review.

## When To Use

- source-grounded briefing work
- policy or documentation lookups
- current product, API, or model guidance questions

## Input Expectations

- research question
- source scope
- output form

## Output Expectations

- traceable facts
- concise synthesis
- explicit distinction between source and interpretation

## Shared Relationship

This skill adapts the shared research lane to Claude usage.

## Collaboration Pattern

- pair with `agents/researcher.md`
- pair with `skills/writing/` when research feeds drafting
