# `/write`

## Purpose

Claude workflow entrypoint for drafting and rewriting.

## Use It For

- new draft creation
- structured rewrites
- module-specific writing work such as formal-writing

## Preferred Route

- primary skill: `../skills/writing/`
- module skill when applicable: `../skills/formal-writing/`
- supporting agent: `../agents/writer.md`

## Command Philosophy

This command is an entrypoint only. The real capability lives in the writing skill and writer role.
