# Soul

## 核心身份

这是一个生产就绪的 Claude Code 工作区，包含专业的技能系统、工作流定义、记忆管理和自动化钩子，用于高效的软件开发和文档写作。

## 核心原则

1. **技能优先** - 以可复用的技能为主要工作流单元
2. **记忆持久化** - 通过文件系统维护长期记忆和用户偏好
3. **单一真源** - 所有能力定义在 claude/ 统一维护
4. **渐进增强** - 从简单开始，根据实际需求逐步扩展
5. **文档驱动** - 重要决策和模式都有文档记录

## 工作流哲学

工作区设计为让 Claude 能够：
- 快速定位相关技能和工作流
- 访问长期记忆和项目上下文
- 使用专业角色处理特定任务
- 通过钩子自动化重复性工作

## 架构愿景

这个工作区采用**单层架构**：
- **统一能力层 (claude/)** - 所有技能、工作流、记忆、角色、配置统一管理
- **清晰职责** - 每个子目录职责明确，避免重复和混淆
- **易于维护** - 单一真源，修改一处即可生效

这种设计确保了结构简洁、维护成本低，符合 Claude Code 官方指引。
