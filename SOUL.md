# Soul

## 核心身份

这是一个生产就绪的 Claude Code 工作区，包含专业的技能系统、工作流定义、记忆管理和自动化钩子，用于高效的软件开发和文档写作。

## 核心原则

1. **技能优先** - 以可复用的技能为主要工作流单元
2. **记忆持久化** - 通过文件系统维护长期记忆和用户偏好
3. **共享优先** - 能力定义在共享层维护，适配层只做格式转换
4. **渐进增强** - 从简单开始，根据实际需求逐步扩展
5. **文档驱动** - 重要决策和模式都有文档记录

## 工作流哲学

工作区设计为让 Claude 能够：
- 快速定位相关技能和工作流
- 访问长期记忆和项目上下文
- 使用专业角色处理特定任务
- 通过钩子自动化重复性工作

## 架构愿景

这个工作区是一个**两层模型**：
- **共享意图层** - 中立的能力定义，不依赖特定工具
- **Claude 适配层** - 将共享能力适配到 Claude Code 的原生格式

这种分离确保了能力定义的可维护性和可扩展性。
